# 🎮 Rock Paper Scissors

A simple and responsive **Rock Paper Scissors** game built using **HTML, CSS, and JavaScript** as a practice project to strengthen front-end development fundamentals.

## 🚀 Features

* ✊ Rock, ✋ Paper, ✌️ Scissors gameplay
* 🤖 Random computer choice
* 🏆 Win, Lose, and Draw detection
* 📊 Live score tracking
* 🎉 Confetti animation for wins
* 📱 Responsive design for desktop and mobile
* ✨ Clean and modern user interface

## 🛠️ Built With

* **HTML5** – Page structure
* **CSS3** – Styling, Flexbox, responsive layout, animations
* **JavaScript (ES6)** – Game logic, DOM manipulation, event handling, score updates

## 📚 What I Practiced

* Semantic HTML
* CSS Flexbox
* Responsive Design
* DOM Manipulation
* Event Listeners
* Functions
* Arrays
* Random Number Generation (`Math.random()`)
* Conditional Statements (`if...else`)
* Updating the UI dynamically
* Working with external libraries (Canvas Confetti)

## 🎯 Purpose

This project was built for learning and practicing core front-end development concepts while improving JavaScript problem-solving skills.
